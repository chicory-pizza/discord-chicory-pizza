# discord.chicory.pizza

Website for [https://discord.chicory.pizza](https://discord.chicory.pizza)

## Development

Requirements:

- [Node v22+](https://nodejs.org)

Steps:

1. git clone
2. npm install
3. npm start

This should open the website at http://localhost:5173
